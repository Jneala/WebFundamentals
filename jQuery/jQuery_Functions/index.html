Below are what I feel are the basic / common jQuery functions.

Play around and see what each one does!

Click

Bind an event handler to the “click” JavaScript event, or trigger that event on an element.

Hide and Show

Hide and then Show the header above.

Toggle

Hide and then Show the header above, but with One button.

Slide Up

Hide the header above with a sliding motion.

Slide Down

Display Slide Up with a sliding down motion. Move to the next button!

Slide Toggle

Display or hide the matched elements with a sliding motion.

Fade Out

Hide Fade Out by fading them to transparent.

Fade In

Display Fade Out by fading them to opaque.

Add/Remove Class

Adds the specified class(es) to each element in the set of matched elements. Here the buttons will add a class used in my css to turn this heading white, then remove it.

Before

Insert content, specified by the parameter, before each element in the set of matched elements.

After


Insert content, specified by the parameter, after each element in the set of matched elements.

Append

Insert content, specified by the parameter, to the end of each element specified. In this case, the p tag.

HTML

Get the HTML contents of the first element in the set of matched elements or set the HTML contents of the element. Change a Link

Attr

Get the attributes of an element, or Set the attributes. Not sure what this is good for, prop seems a better option. Lets use prop to change the checkbox! Go ahead and click the button!

Value

Get the current value of the first element using an identifier such as id, class, tag etc, Or set the value of the element. Here, the button sets the input text box value.

Text

Get the combined text contents of each element in the set of matched elements, including their descendants as in ul, or set the text contents of the matched elements. Like using on click to change a p tag's text, but cannot change actual HTML like the .html function can.
